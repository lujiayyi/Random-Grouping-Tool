# 示例文件说明

## example.csv

这是一个示例CSV文件，包含18个学生的信息。您可以：

1. 直接用Excel打开这个CSV文件
2. 另存为 `.xlsx` 格式
3. 在随机分组工具中上传使用

## 文件格式说明

- **序号**：学生编号（可选）
- **姓名**：学生姓名（必填）
- **性别**：学生性别（可选，如：男、女）
- **年龄**：学生年龄（可选）

## 如何创建自己的Excel文件

1. 打开Excel或任何电子表格软件
2. 在第一行输入表头：序号、姓名、性别、年龄
3. 从第二行开始填入学生信息
4. 保存为 `.xlsx` 或 `.xls` 格式
5. 在工具中上传使用

## 注意事项

- 姓名是必填项，其他可以留空
- 性别留空时会显示为"未知"
- 年龄留空时会显示为"未知"
- 支持任意数量的学生记录
